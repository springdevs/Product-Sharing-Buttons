== Description ==

Share your zest with your friends and others.


## Features

1. Facebook Share
2. Twitter Share
3. LinkedIn Share
4. YouTube Share
5. and Share with any other Social network

== Installation ==

    = Installation from within WordPress =

        1. Visit 'Plugins > 'Add New'.
        2. Search for 'Social Sharing Button'.
        3. Install and activate the 'Social Sharing Button' plugin.

    = Manual installation =

        1. Upload the entire `Social Sharing Button` folder to the `/wp-content/plugins/` directory.
        2. Visit 'Plugins'.
        3. Activate the `Social Sharing Button` plugin.
